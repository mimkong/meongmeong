- 더보기 버튼 클릭 시, BEST 상품 가져오기
- SHOP 메뉴에 커서 hover 시, 카테고리 선택 가능
- <details><summary>시연영상</summary> </details>
### 💡 상세 페이지
- 필수 옵션 선택 후 수량조절 가

### 💡 로그인 / 회원가입
- 모킹API (MSW)를 사용하여 로그인 기능 구현
- 카카오 주소찾기 API를 사용하여 주소 검색 후 간결한 주소 입력

### 💡 최근 본 상품
- 최근 본 상품 장바구니에 담기 및 삭제 가능
